# Theme Editable Journal

根据照片主题自动匹配视觉风格，生成可继续编辑的电子手帐 / Plog。

这个 skill 适合把一组旅行、亲子出行、博物馆、展览、动物园、城市漫游等照片，整理>成适合小红书、朋友圈长图或个人相册记录的可编辑 HTML 手帐。HTML 页面调整完成后，>可以直接导出下载为 PNG 图片，实现所见即所得保存。

## 谁适合使用

- 带孩子逛博物馆、动物园、展览后，想把照片整理成有故事感手帐的家长
- 想做小红书 Plog、旅行路线图、观展记录、亲子研学记录的内容创作者
- 有一组照片，但不确定应该用什么风格、怎么排版、怎么写短文案的人
